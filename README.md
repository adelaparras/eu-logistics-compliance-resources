# EU Logistics Compliance Resources

> A curated collection of resources, regulatory frameworks, and implementation guides for EU logistics digitalization.

## 📋 Contents

### Regulatory Frameworks
- **eFTI Regulation (EU 2020/1056)** – European regulations for freight transport information
- **Digital Transport and Logistics Forum (DTLF)** – EU Commission coordination on logistics digitalization
- **EU Data Spaces** – Data governance and interoperability initiatives

### Key Topics
- Transport modes: road, rail, inland waterways, aviation
- Digital documentation and paperless processes
- Data sovereignty and cross-border compliance
- Authority-operator data sharing frameworks

### Implementation Areas
- Authority access and authentication
- Data verification and immutable record-keeping
- Privacy-preserving information sharing
- Multi-modal transport coordination
